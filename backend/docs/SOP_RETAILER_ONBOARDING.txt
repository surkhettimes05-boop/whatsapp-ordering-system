OVERVIEW
════════════════════════════════════════════════════════════════════════════
Purpose: Enable retailers to place orders via WhatsApp
Timeline: 5-10 minutes per retailer
Owner: Field Operations Manager
Success Rate Target: 95%+ first-time activation

PREREQUISITES
════════════════════════════════════════════════════════════════════════════
□ Retailer has WhatsApp installed on business phone
□ Phone number is active and verified
□ Retailer speaks English or local language
□ Valid business registration (ID required)
□ Bank account details for payment settlement


STEP 1: VERIFICATION & REGISTRATION
════════════════════════════════════════════════════════════════════════════

1.1 Collect Information
    ├─ Business name
    ├─ Owner/Manager name
    ├─ Phone number (WhatsApp enabled)
    ├─ Location/address
    ├─ Business registration number
    ├─ Bank account (IBAN/account number)
    └─ Email (for invoices)

1.2 Create Account in System
    Step: Go to admin.system.com → Retailers → Add New
    Enter:
      ├─ Business Name: [use exact registration name]
      ├─ Phone: +[country code][number]
      ├─ Location: Full address
      ├─ Bank Details: Save securely
      └─ Status: PENDING_VERIFICATION
    
    ✓ Save and note RETAILER_ID (e.g., RTL-20260122-001)

1.3 Verify Bank Details
    Send test deposit: $0.01
    Confirm receipt with retailer
    ✓ Status: VERIFIED
    
    ⚠️ If test fails: Contact Finance team immediately
       Do NOT proceed until bank account verified

STEP 3: CREDIT SETUP
════════════════════════════════════════════════════════════════════════════

3.1 Determine Credit Tier
    
    Volume Category | Daily Limit | Notes
    ─────────────────────────────────────────────────────────
    Micro          | $50         | New retailers (14 days)
    Small          | $200        | Proven history (30+ days)
    Medium         | $500        | Good payment record
    Large          | $1,000+     | VIP/wholesale
    
    Decision factors:
    ├─ Business size
    ├─ Registration legitimacy
    ├─ Bank verification success
    └─ Recommended by existing retailer (if referral)

3.2 Set Credit Parameters
    In system → Retailer Settings → Credit:
    
    ├─ Daily Limit: [amount]
    ├─ Payment Terms: [net_30/net_7/prepay]
    ├─ Credit Buffer: 15% above daily limit
    └─ Auto-topup: Enable/Disable
    
    Status: CREDIT_ACTIVE

3.3 Send Welcome Email
    Include:
    ├─ Credit limit and terms
    ├─ Payment instructions
    ├─ FAQ document
    ├─ Support contact number
    └─ Link to training video

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

PROBLEM: WhatsApp message not delivered
→ SOLUTION:
  1. Verify phone number is correct (including country code)
  2. Ask retailer if WhatsApp is installed
  3. Check if number is in contact list
  4. Try calling, then retry WhatsApp
  5. Escalate to tech team if still failing

PROBLEM: Bank verification fails
→ SOLUTION:
  1. Verify account number is correct
  2. Check bank details format (IBAN vs local)
  3. Confirm account is active at bank
  4. Contact Finance team
  5. May need alternative payment method

PROBLEM: Retailer can't place first order
→ SOLUTION:
  1. Check credit limit is set
  2. Verify status is ACTIVE (not SUSPENDED)
  3. Have retailer send exact product name
  4. Walk through manually in real-time
  5. Check system logs for error messages

PROBLEM: Retailer not responding
→ SOLUTION:
  1. Wait 24 hours
  2. Send follow-up message
  3. Call phone number
  4. Contact through referrer (if applicable)
  5. Mark as inactive after 7 days without response
