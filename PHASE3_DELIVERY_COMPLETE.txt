SUPPORT & QUESTIONS
════════════════════════════════════════════════════════════════════════════════

If staff have questions about procedures:
1. Consult the SOP document
2. Ask their direct manager
3. Manager escalates to Operations Director if unclear
4. New scenarios/edge cases documented for v1.1 update

If you find errors or ambiguities:
1. Document exactly what was unclear
2. How did staff interpret it incorrectly?
3. What would make it clearer?
4. Report to manager for v1.1 update

If you need to deviate from procedures:
1. DO NOT deviate without approval
2. Ask manager for authorization
3. Document the exception
4. Review for inclusion in next SOP version
